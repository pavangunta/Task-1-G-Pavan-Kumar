# Rule-Based Chatbot

## Overview

This project is a simple Rule-Based Chatbot developed using Python. The chatbot responds to predefined user inputs using a dictionary of questions and answers.

The project demonstrates the fundamentals of Natural Language Processing (NLP), conditional logic, and conversational AI.

## Features

* Interactive command-line chatbot
* Predefined responses for common questions
* Handles greetings and basic AI-related queries
* Exit command using "bye"
* Beginner-friendly implementation

## Technologies Used

* Python

## How It Works

The chatbot stores predefined questions and responses in a dictionary. When a user enters a message, the program searches for a matching response and displays it.

If the input is not recognized, the chatbot returns:

"Sorry, I didn't understand that."

## Sample Conversation

You: hello

Chatbot: Hi there! How can I help you?

You: what is ai

Chatbot: AI is the simulation of human intelligence by machines.

You: bye

Chatbot: Goodbye! See you next time.
